# C-COS Studio
### Character Card Operating System

![Version](https://img.shields.io/badge/version-1.0.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)

**Project Overview**
C-COS Studio is a dual-environment creative engine designed for high-fidelity AI character generation and lorebook management. The system integrates visual analysis with psychological frameworks to ensure character consistency and depth. It features a modular architecture built with vanilla JavaScript, HTML5, and CSS3.

**Key Features**

* **Image Analysis**: Extracting physical attributes and aesthetic details from character reference images.
* **Contradiction Discovery**: Identifying internal character conflicts to create grounded, non-linear personas.
* **Psychology Mapping**: Developing deep-seated behavioral traits and internal motivations.
* **Quality Gates**: Automated validation system for dialogue counts, banned word filtering, and section completeness.
* **Voice Fingerprint**: Defining unique speech patterns, vocabulary registers, and physical tells for consistent dialogue.

**C-COS Codex**
* Comprehensive management system for world-building and character lore.
* Support for multiple export formats including SillyTavern and Janitor AI.
* Categorization tools for world data, locations, items, and character relationships.
* Dynamic entry management with adjustable insertion orders and keys.

**Technical Specifications**
* **Frontend**: Responsive UI utilizing custom CSS variables and professional typography (DM Mono and Playfair Display).
* **Logic**: Asynchronous, modular JavaScript engine.
* **Cloud Integration**: Support for OpenAI, Anthropic (Claude), and Google Gemini API formats.
* **Local Support**: Compatible with local backends such as LM Studio, Ollama, and KoboldCpp.
